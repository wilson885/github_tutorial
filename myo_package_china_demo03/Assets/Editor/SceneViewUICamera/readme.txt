GENERAL USAGE NOTES
-------------------
After importing SceneViewUICamera asset to your project, this tool helps you to see UI elements from perpendicular position of parent canvas. 
To turn off this feature, click toggle check-box at left upper corner of current scene view.
You can see UI camera icon at right down corner of scene view when UI mode is active.
